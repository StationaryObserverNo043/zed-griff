# 宇宙語翻訳『ゼド・グリフ』

地球防衛ガジェットとして開発された、宇宙語翻訳アプリです。
地球のひらがな・カタカナを、ゼド星系公用語「ゼド・グリフ」へ変換します。

ゼド・グリフは、日本語の五十音を基盤とした独自の文字体系・発音体系を持つ架空言語です。
GitHub Pagesにて公開中デス。

---

## 主な機能

- 地球語 ⇄ ゼド・グリフ翻訳（暗号化・復号の双方向対応）
- 発音表示
- 文字対応表・言語マニュアル
- モバイル・PWA対応
